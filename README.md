# College Transport Usage Analysis

This website presents a project on College Transport Usage Analysis using DBSCAN with K-Means comparison.

## Project Aim

The aim of this project is to analyze college transport usage patterns, identify high-demand routes, detect unusual travel behavior, and compare clustering results using DBSCAN and K-Means.

## Technologies Used

- HTML
- CSS
- DBSCAN clustering
- K-Means clustering
- Data preprocessing
- CSV dataset analysis

## Website Sections

- Project overview
- Methodology
- DBSCAN analysis
- K-Means comparison
- Key findings
- Dashboard-style results

## Dataset Features

The dataset may include:

- Route name
- Stop name
- Student count
- Travel time
- Distance
- Trip frequency
- Occupancy percentage
- Usage type

## Conclusion

DBSCAN is useful for identifying dense transport usage zones and outliers, while K-Means is helpful for comparing fixed cluster groups.
